### 👨‍💻 About

I'm a data professional who lives at the intersection of **machine learning** and **automation**. I build models that forecast and classify, then wrap them in autonomous systems — scrapers, schedulers, and LLM-powered pipelines — that keep working without me.

- 🔭 **Currently:** data & analytics at Maxima Apparel, building forecasting and automation tooling.
- 🌱 **Focus:** time-series forecasting, ML pipelines, LLM apps, and hands-off automation on GitHub Actions.
- ⚙️ **Philosophy:** if I've done it manually twice, the third time should be a pipeline.

### 🚀 Selected Work

**🤖 Automation systems** — production bots running on scheduled GitHub Actions, no server required:
- **Multi-platform content engine** — scrapes trending topics, generates branded posts with LLMs, and auto-publishes to a LinkedIn Page (via Make.com), Dev.to, and occasion-based greetings.
- **Profile automation suite** — self-running LinkedIn and Naukri updaters that keep profiles active and visible to recruiters.
- **[job-watchdog](https://github.com/tanishq20011430/job-watchdog)** — multi-source job scraper & alerter with semantic + LLM filtering for the Indian market.
- **[sports-ai-agent](https://github.com/tanishq20011430/sports-ai-agent)** — autonomous daily agent that refreshes six sports leagues' standings & odds into a live workbook.

**📊 Machine learning & data science:**
- **XGBoost demand forecasting** — production forecasting pipeline with engineered time-series features, served via API.
- **LearnLinker** — a Next.js AI data-analyzer: upload a CSV/Excel and get instant insights via Gemini.
- **End-to-end ML pipelines** — EDA → model → deployment, plus time-series and classification projects.

> Some of the above live in private repos — happy to walk through the code or architecture on request.
